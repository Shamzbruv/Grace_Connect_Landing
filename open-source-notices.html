Open Source Notices

Notice wording for open-source packages and third-party libraries used by Grace Connect.

Last Updated: June 16, 2026

1. Open Source Components

Grace Connect may use open-source software packages and SDKs, including Flutter/Dart packages and third-party libraries for Supabase, Firebase, Google services, maps, notifications, images, files, audio, and app utilities.

2. License Compliance

Each open-source component remains subject to its own license terms. Grace Connect should preserve required copyright notices, attribution notices, license files, and disclaimers.

3. In-App License Screen

The mobile app should include an “Open Source Licenses” or “Licenses” screen in Settings or Legal. Flutter apps may use the built-in license page or a custom screen that lists package names and licenses.

4. No Warranty From Open Source Authors

Open-source components are generally provided without warranties. Grace Connect users should not contact open-source authors for Grace Connect support.
